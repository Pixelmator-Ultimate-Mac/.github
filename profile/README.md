## Overview

<a href="#">Pixelmator Pro</a> is a professional image editor built exclusively for Mac that combines photo retouching, graphic design, painting, and RAW processing in a single application powered by machine learning tools trained on millions of images.

The <a href="#">layer-based editing system</a> supports pixel layers, vector layers, text layers, group layers, and effect layers in any combination. Apply effects and adjustments as non-destructive layer effects that can be modified or removed without affecting the underlying image data. <a href="#">Layer styles</a> add shadows, outlines, fills, and gradients to any layer type through a consistent visual interface.

<a href="#">ML-powered tools</a> accelerate tasks that would otherwise require extensive manual work. ML Denoise removes noise from photos while preserving fine detail better than traditional noise reduction algorithms. ML Super Resolution enlarges images while generating detail rather than simply scaling pixels. <a href="#">ML Background Removal</a> isolates subjects from backgrounds with a single click, handling complex edges like hair and transparent objects accurately.

The <a href="#">RAW processing pipeline</a> handles files from every major camera manufacturer with full control over exposure, white balance, highlight recovery, shadow lifting, lens correction, and noise reduction. <a href="#">Non-destructive RAW editing</a> preserves the original file and all adjustments as an editable sidecar, allowing any parameter to be revisited without quality loss. Color profile support ensures accurate color from camera through to output.

## Additional Information

Pixelmator Pro is one of the clearest examples of what prioritizing a single platform produces. The machine learning features are not marketing additions — they work better than equivalent tools in cross-platform editors because the models were trained and deployed using Apple's Neural Engine and Core ML stack rather than a generic third-party inference pipeline. ML Super Resolution is the specific case that converts photographers: enlarging a shot that would have been unusable at print size into something that holds up at 2x is the kind of capability that changes which photos get used rather than discarded.
